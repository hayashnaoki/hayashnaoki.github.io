# Language learning app

2026 \| Web \| LLM

I always wondered it would be much more interesting if I can learn languages with contents I am interested in, such as 3D printing, electronics design. This is a prototype of an AI-powered language learning tool that converts any article or any language into simplified, multilingual (currently Chinese) learning materials.

MVP: [https://lingua-gen--hayashnaoki.replit.app/](https://lingua-gen--hayashnaoki.replit.app/)
