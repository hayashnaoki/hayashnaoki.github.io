### 3D physical cellular automata

Logic
- No central control, decentralized logic
- Read states of neighboring cells, state of each cell is determined
- LEDs in the cells light on or off according to the state (alive/dead)
- Repeat until reaching stable state

Structure
- Powered from a base through cells
- Modular physical structure for easy assembly
- Each cell equip minimal logic circuit but no MCU
