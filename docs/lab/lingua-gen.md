# Lingua-gen

2026 \| Web \| LLM

I've always wondered why there isn't an app that lets you learn a language using content you're interested in. This is an AI-powered language learning tool that converts any article or any language into simplified, multilingual (currently Chinese) learning materials.

MVP: [https://lingua-gen--hayashnaoki.replit.app/](https://lingua-gen--hayashnaoki.replit.app/)
